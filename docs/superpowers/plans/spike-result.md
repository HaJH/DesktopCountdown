# 스파이크 결과: WorkerW 레이어드 자식 창

Task 1 (`.superpowers/sdd/task-1-brief.md` Step 5)에서 정의한 4가지 성공 기준의 검증 결과.

**이 파일은 자동화 에이전트가 채울 수 없다.** 시각적 확인이 필요하므로, 사람이 직접
`cargo run --bin spike_layered`을 실행하고 바탕화면을 눈으로 확인한 뒤 아래 각 항목의
체크박스와 "결과" 줄을 채워야 한다. 넷 다 통과해야 Task 2로 진행할 수 있고, 하나라도
실패하면 여기서 멈추고 설계 §9(벽지 직접 합성)로 전환할지 판단해야 한다 (Task 8·10·11
재작성 필요).

## 실행 방법

```
cargo run --bin spike_layered
```

`Win+D`로 바탕화면을 보이게 한 뒤 아래 항목을 확인한다. 종료는 `Ctrl+C`.

      빨갛게 진해진다.

      사각형 위치에 바탕화면 아이콘이 있으면 아이콘이 사각형 위에 그려진다. (없으면
      아이콘을 그 자리로 잠깐 옮겨 확인한다.)

- [ ] **3. 다른 창에 가려지는가**
      아무 창이나 띄우면 사각형이 그 창에 가려진다.
